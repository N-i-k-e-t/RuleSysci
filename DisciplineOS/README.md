# DisciplineOS

Psychology-First Trading Journal & Discipline System.

## Project Setup

1.  **Install Dependencies**: `npm install` (already done)
2.  **Start the Server**: `npx expo start`
3.  **Run on Android**: `npx expo run:android` or scan QR code with Expo Go app.
4.  **Run on iOS**: `npx expo run:ios` (Mac only) or scan QR code.

## Tech Stack

*   React Native + Expo SDK 54+
*   NativeWind (TailwindCSS)
*   Zustand (State Management)
*   Expo SQLite (Local Database)
*   React Navigation v7
